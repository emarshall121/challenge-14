# Challenge 14 - Model View Controller

## Overview
The purpose of this assignment was to build a CMS style blog that can be used to publish, read, and comment on blog posts. 

## Technologies Used
- Express.JS
- Handlebars.JS
- Nodemon
- Express Session
- Heroku
- Sequelize
- MySQL2
- Bcrypt package
